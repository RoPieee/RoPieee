The **RoPieee Network Installer** simplifies the installation process. It automatically detects your Raspberry Pi model and installs the corresponding, latest stable RoPieee release, ensuring ease of use.

So installation of RoPieee becomes as simple as this:

1. Download the Network Installer [compressed](https://image.ropieee.io/ropieee_net_installer-20260410.3476.bin.xz) and [uncompressed](https://image.ropieee.io/ropieee_net_installer-20260410.3476.bin)
2. Flash this image onto the microSD card.
3. Insert the MicroSD card into your Raspberry Pi.
4. Power it on.

> [!WARNING]
> The Network Installer will not work on the Orchard Audio PecanPi+ device.
